# LDAL

Linux 设备抽象层


## Features

- 依赖于设备节点（如 /dev/ttyUSB0）
- 线程安全，支持多线程并发访问
- 支持注册多个同类型设备
- 异步操作
- 接口简单
- 控制任务线程化
- 支持中断调用
