# LDAL

Linux device abstraction layer 


## Features

- Dependent on device node
- Thread safe
- Support to register multiple devices of the same type



## Support device

| Class  | Description                                                | Comment |
| ------ | ---------------------------------------------------------- | ------- |
| Serial | Device node like /dev/ttyUSB0, including RS-232 and RS-485 |         |
| GPIO   |                                                            |         |
| RTC    |                                                            |         |
